What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the look and formatting of a document written in markup language such as HTML. It is used to define styles, layout, and positioning of HTML elements on a web page.

CSS is based on the concept of cascading, which means that multiple styles can be applied to the same element, and the most specific style takes precedence over the others. This allows for a lot of flexibility in styling web pages.

Introduction

CSS uses a simple syntax that consists of a selector and a declaration block. The selector is used to specify the HTML element that you want to style, and the declaration block contains one or more properties that define the style of the selected element.

Here is an example of the syntax

selector { property: value; }

In this example, selector is the HTML element you want to style, property is the CSS property you want to set, and value is the value you want to assign to that property. Multiple properties can be included in a declaration block separated by a semicolon.

CSS Selectors

CSS selectors are used to select and style specific HTML elements. There are several types of selectors, including:

  • Type selector: Selects all elements of a given type, such as `h1`, `p`, or `img`.
  • Class selector: Selects all elements with a given class name, using the . followed by the class name, such as `.my-class`.
  • ID selector: Selects a single element with a given ID, using the # followed by the ID name, such as `#my-id`.
  • Descendant selector: Selects all elements that are descendants of a given element, such as `div p`.
  • Child selector: Selects all direct children of a given element, such as `div > p`.
  • Adjacent sibling selector: Selects the next sibling of a given element, such as `h1 + p`.
  • General sibling selector: Selects all siblings of a given element, such as `h1 ~ p`.
  • Attribute selector: Selects elements based on their attributes, such as `a[href="https://www.google.com"]`.
CSS Properties

CSS properties are used to set the style of HTML elements. There are hundreds of properties available in CSS, but some of the most commonly used properties include:

Box Model

Here's an example of how to set the padding, border, and margin of an element:

.my-class { padding: 10px; border: 1px solid black; margin: 20px; }
  • Width/Height: Sets the width and height of an element.
  • Padding: Sets the padding area around the content inside an element.
  • Margin: Sets the margin area around an element.
  • Border: Sets the border around an element.

Texts and Fonts

  • Color: Sets the color of text or background.
  • Fonts: Sets the font family, size, weight, and style.
  • Text-align: Sets the horizontal alignment of text.
  • Line-height: Sets the height of a line of text.
  • Text-decoration: Sets decorations such as underline, overline, and line-through.
  • Text-transform: Sets the capitalization of text.
  • Letter-spacing: Sets the spacing between letters.
  • Word-Spacing: Sets the spacing between words.

Display and Positioning

  • Display: Sets how an element is displayed on the page.
  • Position: Sets the positioning of an element.
    • .my-class { position: static; }
  • Z-index: Sets the stacking order of elements.
  • Overflow: Sets what happens when content overflows an element.
  • Visibility: Sets the visibility of an element.

Background and Images

  • Background-color: Sets the background color of an element.
  • Background-image: Sets the background image of an element.
  • Background-repeat: Sets how a background image is repeated.
  • Background-position: Sets the position of a background image.
  • Background-size: Sets the size of a background image.

Transitions and Animations

  • Transition: Sets the transition effect between two states of an element.

    • Here's an example of how to create a transition on a hover effect:

    button { background-color: blue; transition: background-color 0.5s ease; } button:hover { background-color: red; }
  • Animation: Sets the animation effect of an element.

    • Here's an example of how to create an animation on a loading spinner:

    .spinner { width: 50px; height: 50px; border-radius: 50%; border: 5px solid #ccc; border-top-color: #333; animation: spin 1s infinite linear; } @keyframes spin { to { transform: rotate(360deg); } }
CSS Units

CSS uses different units to define measurements for various properties such as size, spacing, and positioning. Some of the most commonly used units include:

  • Pixel(px): A pixel is a unit of measurement that is used to define the size and position of elements on a web page. One pixel is equal to one dot on the screen.
  • Percentage(%): A percentage is a unit of measurement that is used to define the size and position of elements relative to the size of their containing element.
  • EMs(em): An em is a unit of measurement that is based on the size of the font being used. One em is equal to the font size of the element.
  • REM(rem): A rem is a unit of measurement that is based on the font size of the root element. This is useful for creating consistent layouts across different devices and screen sizes.
  • Viewport(vh,vw): Viewport units are relative to the size of the browser window. One viewport height (vh) is equal to 1% of the height of the browser window, and one viewport width (vw) is equal to 1% of the width of the browser window.
CSS Layouts

CSS layouts are used to arrange HTML elements on a web page. There are several types of layouts, including:

  • Fixed layout: The elements on the page have fixed positions and do not move even if the browser window is resized.
  • Fluid layout: The elements on the page adjust their positions and sizes according to the size of the browser window.
  • Responsive layout: The layout adjusts to different screen sizes and orientations, using media queries to adjust styles and layouts for different devices.
  • Grid layout: The layout is based on a grid system, allowing for precise positioning and alignment of elements.
  • Flexbox layout: The layout is based on a flexible box system, allowing for flexible and dynamic layouts that adapt to different screen sizes and orientations.
CSS Frameworks

CSS frameworks are pre-built sets of CSS styles and components that can be used to create consistent and responsive layouts quickly and easily. Some popular CSS frameworks include Bootstrap, Foundation, and Bulma.

  • Bootstrap

    • Bootstrap is known for popularizing the focus on responsive design among web developers. It promoted the now-ubiquitous concept of mobile-first and provided the right tools for its easy implementation. It did so by introducing a grid – partitioning the screen into columns (invisible to the end user’s eye).

    With Bootstrap, developers don’t have to build separate projects around adjusting a site for smaller screen sizes. They just need to incorporate the necessary Bootstrap classes, and the design adjusts itself.

  • Tailwind CSS

    • Tailwind CSS’s official documentation describes it as a “utility-first CSS framework” that comes with classes equipped to build custom UI designs directly in the users’ markup. It is handy to implement inline styling to rustle up a stunning UI without writing any CSS.

    One of the most popular utility CSS libraries in existence, Tailwind CSS offers significant benefits for web design. Interestingly, it took Adam Wathan (the creator of Tailwind) to convince the world why utility-based CSS trumps semantic CSS. But eventually, enough developers believed him enough to start using Tailwind, and now it is used frequently enough to be on lists like this.

  • Foundation

    • Foundation calls itself “The most advanced responsive front-end framework in the world.” This responsive front-end framework provides a grid, HTML, SASS, and CSS UI elements, templates, and code that covers navigation, buttons, typography, forms, etc. It also comprises optional functionalities offered by JavaScript extensions.

    Formerly managed by ZURB, Foundation is an open-source project that volunteers have run since 2019. It also operates with a mobile-first approach and is particularly useful for building large web apps requiring a design host.

  • Bulma

    • Bulma is an open-source, responsive CSS framework based on Flexbox. It comes with an impressive range of built-in features that facilitate faster turnaround and minimal CSS coding by hand. It implements tiles to create Metro-style grids, resulting in smooth page layouts. Additionally, users can import only the elements they will use, simplifying the process even further.

    Given that Bulma’s source code is free for download, users can extend its functionality as they see fit. It provides a no-nonsense, CSS-only methodology (no JavaScript components) and visually appealing defaults to get started with.

    Bulma has been highly adopted within the Laravel community, which has helped with its increasing popularity.

  • Skeleton

    • Skeleton doesn’t even call itself a framework but rather a “dead simple, responsive boilerplate.” Its minimalism is beyond question: it has only 400 lines of source code!

    This lightweight tool is built to create CSS elements compatible with both larger screens and mobile devices. It contains all standard components for responsive design and splits the page into multiple 12-column grids with a maximum width of 960px – something that fits small, medium, and large screens. Of course, if required, the max-width can be changed with a single line of CSS code. The syntax is quick to implement, simple, and makes putting together responsive design a relatively effortless task.

Conclusion

CSS is a powerful tool for styling and formatting HTML documents. With a solid understanding of CSS syntax, selectors, properties, units, and layouts, you can create beautiful and responsive web pages that look great on any device.
